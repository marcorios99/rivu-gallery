# Changelog

All notable changes to Rivu Gallery will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

### Added
- Added Rivu Gallery branding and launcher icon.
- Added bottom navigation for collection, videos, favorites, and albums.
- Added recycle bin access from the main overflow menu.
- Added map preview support in media properties.
- Added support for keeping managed empty albums visible.

### Changed
- Reworked the About screen for Rivu Gallery.
- Updated the default launcher icon color to blue.
- Changed the default media grouping to Date Taken, daily, descending.
- Improved folder rename feedback and fallback behavior to reduce perceived freezes.
- Stabilized album refresh behavior to reduce visual jumping during scans.
- Updated Fastlane package name to `com.riosstudio.rivugallery`.

### Removed
- Removed Fossify-specific store metadata translations except English and Spanish.
- Removed Fossify community links from the in-app About experience.

## Upstream History

Rivu Gallery started as an independent fork of Fossify Gallery.

Earlier upstream history is available in the Fossify Gallery repository:
