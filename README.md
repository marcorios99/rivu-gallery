# Rivu Gallery

Rivu Gallery is a simple, private Android gallery for local photos and videos.

It is designed around a clean album experience, quick access to collection, videos,
favorites and recycle bin, and practical details such as grouped media, photo
properties, location metadata, and offline-first browsing.

## Status

Rivu Gallery is currently in active development by Rios Studio.

The app is based on Fossify Gallery and is being adapted with its own branding,
interface choices, and product direction. Some assets, package names, metadata,
and references may still be in transition while the fork is being prepared as an
independent project.

## Features

- Local photo and video gallery
- Albums, collection, videos, favorites, and recycle bin views
- Recycle bin with automatic cleanup after 30 days
- Media grouping by date
- Photo/video properties
- EXIF and GPS metadata support
- Offline-first experience
- No ads
